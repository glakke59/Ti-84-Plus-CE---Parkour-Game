# Ti-84 Plus CE - Parkour Game
A 3D wireframe parkour game for the TI-84 Plus CE graphing calculator.

### Features
* **Real-time 3D Engine**: High-performance wireframe rendering using 3D-to-2D projection.
* **First-Person Perspective**: Fully immersive movement with jumping and rotation.
* **Optimized for TI-84**: Fine-tuned logic to maintain high frame rates on calculator hardware.

### Controls
* **Movement**: [▲] Up / [▼] Down arrows to walk.
* **Rotation**: [◄] Left / [►] Right arrows to spin the camera.
* **Action**: [ENTER] to jump.
* **Exit**: [CLEAR] to quit the program.

### Installation
1. Download the recent version of the CEPARKO Zip folder in [releases]
2. Connect your calculator and open **TI Connect™ CE**.
3. Drag and drop the .8xp file into the calculator's memory or archive.
4. On your calculator, press [PRGM], select the game, and press [ENTER] to start jumping!

Troubleshooting:

If the program doesn't run, you may need to install Artifice Shell since the recent OS versions require a shell to execute assembly/C-based programs.
